# Technology Stack

## Core Framework & Runtime
- **Framework:** [Astro 5](https://astro.build/) - Utilizing Static Site Generation (SSG) for high-performance delivery.
- **Language:** TypeScript/JavaScript (ES Modules).

## Frontend UI
- **Library:** [React 19](https://react.dev/) - Used for complex interactive components like the Budget Wizard and Status Form.
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/) - Utility-first CSS framework for rapid and consistent UI development.
- **Icons:** [Lucide React](https://lucide.dev/) - Modern and consistent icon set.

## Build & Optimization
- **Bundler:** Vite (Integrated with Astro).
- **SEO:** Astro Sitemap integration and manual JSON-LD Schema Markup for local business visibility.
- **Images:** Astro's built-in image optimization for WebP delivery.

## Deployment & Infrastructure
- **Hosting:** Hostinger (Static site deployment to `public_html`).
- **CI/CD:** GitHub Actions (configured in `.github/workflows/`).
